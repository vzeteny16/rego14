# Regő OS

Személyes, game-like születésnapi ajándék webapp Regőnek.

## Megnyitás

Nyisd meg az `index.html` fájlt böngészőben. A jelenlegi verzió statikus, nincs szükség buildre vagy backend szerverre.

## Jelenlegi flow

- Regő OS intro / mission briefing
- program választás
- kaja választás
- utána anime / sorozat / játék választás
- opcionális extra reward
- végső összegzés és rögzítés

## Későbbi irány

- tartalmak finomítása
- saját ikon / app ikon
- Vercel deploy
- GitHub repository inicializálás és feltöltés
